# Subscription Management Bot (FinTech)

A scalable Telegram bot solution for managing private channel access via paid subscriptions.

## Key Features
- **Payment Integration**: Seamless integration with YooKassa API for real-time transaction processing.
- **Access Control**: Automated entry/removal logic for users in private Telegram groups based on subscription status.
- **Automated Billing**: Logic for monitoring subscription expiration and renewal alerts.

## Tech Stack
- **Language**: Python
- **Bot Framework**: aiogram
- **Database**: PostgreSQL with Alembic for migrations
- **API**: YooKassa API integration
